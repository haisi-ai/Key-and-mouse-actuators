# 🖱️ 键鼠执行器 v2.0

[![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)](https://www.python.org/)
[![PyQt5](https://img.shields.io/badge/PyQt5-5.15+-green.svg)](https://www.riverbankcomputing.com/software/pyqt/)

一个功能强大的键鼠自动化操作工具，支持录制、脚本编辑、多种操作类型，让您的重复性工作自动化。

## ✨ 特性

### 🎯 丰富的操作类型
- **键盘操作** - 键盘输入、按键按下/释放、快捷键
- **鼠标操作** - 点击、双击、移动、拖动、滚动
- **窗口操作** - 移动窗口、调整大小、激活窗口
- **程序控制** - 启动程序、关闭程序
- **流程控制** - 等待、循环执行

### 🎙️ 录制功能
- 实时录制鼠标点击、滚动和键盘按键
- 可选择录制鼠标或键盘操作
- 自动将录制内容转换为操作序列

### 🎨 现代化界面
- 深色/浅色双主题自由切换
- 双面板布局，操作便捷
- 实时鼠标坐标显示
- 进度条和状态提示

### 💾 脚本管理
- 保存/加载脚本（.kms格式）
- 操作列表的增删改查
- 操作顺序调整（上移/下移）
- 操作启用/禁用

### ⌨️ 全局快捷键
| 快捷键 | 功能 |
|--------|------|
| `F8` | 开始/停止执行 |
| `F9` | 暂停/继续执行 |
| `F10` | 开始/停止录制 |
| `F1` | 打开帮助文档 |

### 🔧 辅助工具
- **系统进程查看器** - 查看所有运行中的进程，双击复制进程名
- **活动窗口查看器** - 查看所有窗口信息，双击复制窗口标题
- **系统托盘支持** - 最小化到托盘，后台运行

## 📋 系统要求

- Windows 7 / 8 / 10 / 11
- Python 3.7 或更高版本
- 依赖库（见下方安装说明）

## 🚀 快速开始

### 方式一：直接运行（需要Python环境）

1. **安装依赖**
```bash
pip install pyautogui pygetwindow psutil PyQt5 pynput requests
```

2. **运行程序**
```bash
python 键鼠执行器.py
```

### 方式二：打包成exe（无需Python环境）

1. **安装PyInstaller**
```bash
pip install pyinstaller
```

2. **打包程序**
```bash
pyinstaller --onefile --windowed --name "键鼠执行器" --icon=icon.ico 键鼠执行器.py
```

## 📖 使用教程

### 基础操作

1. **添加操作**
   - 选择操作类型（键盘输入、鼠标点击等）
   - 填写参数（坐标、按键、文本等）
   - 点击"添加操作"按钮

2. **编辑操作**
   - 双击列表中的操作
   - 修改参数和描述
   - 点击"更新操作"

3. **录制操作**
   - 切换到"录制"选项卡
   - 点击"开始录制"或按F10
   - 执行鼠标/键盘操作
   - 点击"停止录制"

4. **执行脚本**
   - 设置循环次数（0=无限）
   - 设置操作间隔时间
   - 点击"开始"或按F8

### 操作参数说明

#### 键盘输入
```
参数格式: 要输入的文本
示例: Hello World
说明: 模拟打字输入
```

#### 鼠标点击
```
参数格式: X坐标,Y坐标,按键,点击次数
示例: 500,300,left,1
按键: left(左键) / right(右键) / middle(中键)
```

#### 鼠标移动
```
参数格式: X坐标,Y坐标,持续时间
示例: 800,400,0.5
说明: 平滑移动到指定位置
```

#### 快捷键
```
参数格式: 按键1+按键2
示例: ctrl+c / alt+tab / win+r
```

#### 等待
```
参数格式: 秒数
示例: 2.5
说明: 暂停执行指定时间
```

#### 移动窗口
```
参数格式: 窗口标题,X坐标,Y坐标
示例: 记事本,100,100
说明: 移动窗口到指定位置
```

#### 调整窗口
```
参数格式: 窗口标题,宽度,高度
示例: 记事本,800,600
```

#### 启动程序
```
参数格式: 程序路径
示例: notepad.exe
      C:\Program Files\app.exe
```

#### 关闭程序
```
参数格式: 进程名称
示例: notepad.exe
说明: 可通过"工具"→"系统进程"查看进程名
```

### 获取窗口标题和进程名

1. 点击菜单栏"工具" → "系统进程"
2. 找到目标程序，双击进程名即可复制
3. 点击"工具" → "活动窗口"
4. 找到目标窗口，双击标题即可复制

## 🛠️ 常见问题

### Q: 为什么有些操作没有生效？
**A:** 请确保：
- 目标窗口处于激活状态
- 以管理员身份运行（某些程序需要）
- 检查参数格式是否正确

### Q: 如何实现无限循环？
**A:** 将"次数"设置为0即可。

### Q: 录制的操作太多怎么办？
**A:** 可以在操作列表中删除不需要的操作，或手动编辑参数。

### Q: 如何调整执行速度？
**A:** 修改"间隔"参数，数值越大执行越慢。

### Q: 脚本文件可以分享给其他人吗？
**A:** 可以，.kms文件是JSON格式，但需要注意对方电脑上的程序路径是否一致。

### Q: 程序占用CPU高吗？
**A:** 正常情况下CPU占用很低，执行时会略有升高。
