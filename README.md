# 键鼠执行器

一个基于 PyQt5 开发的键鼠操作自动化工具，支持录制和执行一系列鼠标键盘操作，适用于自动化测试、日常办公自动化等场景。

## 功能特性

- **鼠标操作**：移动、点击、拖动、滚动
- **键盘操作**：键盘输入、模拟按键、快捷键组合
- **窗口控制**：移动窗口位置、调整窗口大小
- **程序管理**：启动/关闭指定程序
- **操作列表管理**：添加、删除、修改、排序操作项
- **配置持久化**：保存/加载操作列表为 JSON 文件
- **循环执行**：支持设置执行次数（无限循环/指定次数）
- **全局快捷键**：支持自定义启动/停止快捷键（默认 F8 启动，F10 停止）
- **日志系统**：实时显示执行状态和错误信息
- **版本检查**：自动检查 GitHub 上的更新版本

## 操作类型说明

| 操作类型 | 参数示例 | 说明 |
|---------|---------|------|
| 键盘输入 | `hello world` | 模拟键盘输入文本 |
| 模拟按键 | `a,2,0.5` | 按键名, 次数, 间隔(秒) |
| 按快捷键 | `ctrl+s` | 支持组合键 |
| 鼠标点击 | `left,2,0.5` | 按键(left/right/middle), 次数, 时间 |
| 鼠标移动 | `500,300` | X坐标, Y坐标 |
| 鼠标拖动 | `500,300,1` | X坐标, Y坐标, 时长(秒) |
| 鼠标滚动 | `200` | 正数上滚，负数下滚 |
| 停留时间 | `2.5` | 等待时间(秒) |
| 移动窗口 | `100,100` | 窗口左上角目标坐标 |
| 窗口大小 | `800,600` | 目标宽度, 高度 |
| 启动程序 | `notepad.exe` 或完整路径 | 启动指定程序 |
| 关闭程序 | `notepad.exe` | 关闭指定进程名 |

## 安装说明

### 环境要求
- Python 3.6 或更高版本
- Windows / Linux / macOS

### 安装依赖

```bash
pip install pyautogui pygetwindow pynput psutil requests PyQt5
```

### 运行程序

```bash
python 键鼠执行器v1.0.py
```

## 使用指南

### 基础操作流程

1. **选择操作类型**：从下拉框选择要添加的操作
2. **输入参数**：根据提示输入对应的操作参数
3. **添加操作**：点击「添加」按钮将操作加入列表
4. **管理操作**：使用「上移/下移」调整顺序，「修改」编辑已有操作，「删除」移除操作
5. **设置执行参数**：配置循环间隔时间和执行次数
6. **启动执行**：点击「启动」按钮或按快捷键 F8 开始执行

### 全局快捷键

- **F8**：开始执行操作
- **F10**：停止执行操作

可在「菜单」→「设置」中修改快捷键。

### 文件操作

- **保存**：将当前操作列表保存为 JSON 文件
- **加载**：加载之前保存的操作列表
- **清空**：清空当前操作列表
- **日志导出**：菜单中可保存日志内容

### 辅助功能

- **工具菜单**：查看系统进程列表、活动窗口列表
- **帮助菜单**：查看使用说明、检查更新、关于信息
- **实时坐标显示**：窗口顶部实时显示鼠标当前位置

## 注意事项

1. **权限问题**：部分操作（如关闭程序）可能需要管理员权限
2. **窗口操作**：移动窗口和调整窗口大小仅对当前活动窗口生效
3. **程序启动**：启动程序时如果已运行则不会重复启动
4. **快捷键冲突**：自定义快捷键可能与系统或其他软件快捷键冲突
5. **参数格式**：参数中的逗号请使用英文逗号

## 常见问题

**Q: 为什么鼠标拖动操作不生效？**  
A: 请确保添加了持续时间参数，拖动太快可能导致操作无效。建议设置为 0.5-1 秒。

**Q: 如何停止无限循环执行？**  
A: 按 F10 快捷键或点击窗口上的停止按钮。

**Q: 程序启动提示"操作列表为空"？**  
A: 请先添加操作后再点击启动。

**Q: 移动窗口/调整大小无效？**  
A: 请确保操作执行时目标窗口处于活动状态（鼠标点击过该窗口）。
